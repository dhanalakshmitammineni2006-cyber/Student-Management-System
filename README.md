🎓 Student Management System
A simple Student Management System built using Flask and SQLite with user authentication and CRUD operations.
Features
  Register & Login
  Add Student
  View Students
  Edit Student
  Delete Student
  API Endpoint → /api/students
  Modern Responsive UI
Tech Stack
  Python
  Flask
  SQLite
  HTML & CSS
  ⚙️ How to Run
1️.Create Virtual Environment
  python -m venv venv
Activate it:
  venv\Scripts\activate
2️.Install Flask
pip install flask
3.Run App
  python app.py
4.Open in browser:
  http://127.0.0.1:5000/
